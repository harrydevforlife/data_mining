d.(3 im) S dng thuc tnh Species lm nhn cm tht s, hy tnh v so snh Precison, Recall, v F-measure ca kt qu gom cm (theo cng thc c nh ngha nh bn di) khi dng kmeans, hclust v dbscan.

Gi s tp d liu \(D\) c \(n\) phn t \(x_i\) c phn hoch thnh \(p\) nhm ( y ng vi s loi). Gi \(y_i \in \{1, 2, , p\}\) l nhm tht s (ground-truth labels) cho mi phn t. Ground-truth clustering c cho bi \(T = \{T_1, T_2, \cdots , T_p\}\), vi \(T_j\) bao gm tt c cc phn t c nhn \(j\), ngha l, \(T_j = \{ x_i \in D | y_i = j \}\). Mt khc, gi \(C = \{ C_1, C_2, \cdots, C_k \}\) l mt kt qu gom cm ca \(D\) thnh \(k\) cm (cluster), qua mt thut ton gom cm no , v \(\hat{y_i} \in \{ 1, 2, \cdots, k \}\) l cluster label cho \(x_i\). Ta s xem \(T\) l mt phn hoch chun (ground-truth partitioning) v mi \(T_i\) l mt phn vng (partition). Ta gi \(C\) l mt kt qu gom cm (clustering), vi mi \(C_i\) l mt cm (cluster). Gi s ground truth l bit trc, mt thut ton gom cm s thc hin gom cm trn \(D\) vi s cm chnh xc, tc vi \(k = p\). Tuy nhin, gi tnh tng qut, ta cho php \(k \ne p\).

Contingency table of clustering results
Clusters/Species T1 T2 Tp
C1 n11 n12 n1p
C2 n21 n22 n2p
Ck nk1 nk2 nkp

Cc o nh gi kt qu gom cm c gng nm bt mc m cc phn t t cng mt phn vng (partition) xut hin trong cng mt cm (cluster) v mc m cc phn t t cc phn vng (partition) khc nhau c nhm thnh cc cm (cluster) khc nhau. Nhng o ny da trn \(k \times p\) contingency table \(N\) (xem Table \(\ref{tab:contingency_table}\)) c thnh lp da vo mt kt qu gom cm (clustering) \(C\) v mt phn hoch chun (ground-truth partitioning) \(T\), c nh ngha nh sau:

\[N(i, j) = n_{ij} = |C_i \cap T_j|\]

  • \(Recall\) l t l i tng cng loi c gn cng cm.

  • \(Precision\) l t l i tng c gn cng cm thuc cng loi.

  • \(F{\text -}measure\) l mt o cn bng gia \(Precision\) v \(Recall\) v c tnh bng trung bnh iu ha gia \(Precision\) v \(Recall\). y l mt o thng c s dng so snh cc thut ton gom cm vi nhau.

Cc o \(Precision\), \(Recall\), v \(F{\text -}measure\) c tnh t Table \(\ref{tab:contingency_table}\) dng cc cng thc sau:

\[ \begin{equation} precision = \frac{{\sum\limits_{i = 1}^k {\mathop {{\rm{max}}}\limits_{j \in \left\{ {1, \ldots p} \right\}} \{n_{ij}\} } }}{{\sum\limits_{i = 1}^k {\sum\limits_{j = 1}^p {n_{ij} } } }} \end{equation} \]

\[ \begin{equation} recall = \frac{{\sum\limits_{j = 1}^p {\mathop {{\rm{max}}}\limits_{i \in \left\{ {1, \ldots ,k} \right\}} \{n_{ij}\} } }}{{\sum\limits_{i = 1}^k {\sum\limits_{j = 1}^p {n_{ij} } } }} \end{equation} \]

\[ \begin{equation} \begin{split} F{\text -}measure = \frac{{2 \cdot precision \cdot recall}}{{precision + recall}} \end{split} \end{equation} \]
