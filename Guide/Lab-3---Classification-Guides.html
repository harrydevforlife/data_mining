2.3. nh gi m hnh

nh gi m hnh ta s to confusion matrix v tnh cc o accuracy, precision, recall, f-measure t kt qu d on ca m hnh v nhn lp thc s trn tp d liu test. Bn di l mt v d v cch tnh o accuracy. Hm accuracy nhn vo hai vector l \(actual\) (nhn tht s) v \(predicted\) (nhn c d on bi m hnh) v tr ra gi tr accuracy.

confusion_matrix <- function(predicted, actual) {
  return(table(predicted, actual))
}

accuracy <- function(cm) {
  return(sum(diag(cm))/sum(cm))
}
actual <- testSet$Species
cm <- confusion_matrix(knnmodel.fit, actual)
cm

## predicted    setosa versicolor virginica

##   virginica       0          1        17

## [1] 0.94

2.4. nh gi m hnh dng phng php cross validation

Phn sau y ta s tin hnh nh gi theo phng php cross validation (thay v phng php holdout nh trn) vi cc gi tr k (trong k-nn) khc nhau tm ra gi tr k tt nht. u tin ta s to cc fold, sau ta s th nghim trn cc fold ny vi cc gi tr k khc nhau.

set.seed(1111)
nfolds <- 3 # s fold
ids <- 1:nrow(iris)
index <- sample(ids) # xao tron ngau nhien ids
fold <- rep(1:nfolds, each = nrow(iris)/nfolds)
folds <- split(index, fold) # tao mot danh sach voi cac index cho moi fold
ks <- seq(1, 19, 2) # k = 1, 3, 5, ..., 19
accs <- matrix(0, nrow = length(ks), ncol = nfolds + 1)
for (k in 1:length(ks)) {
  s = 0
  for (i in 1:nfolds) {
    t <- kknn(Species ~ ., 
              train = iris[-folds[[i]], ], 
              test = iris[folds[[i]], ], 
              k = k, 
              distance = 2, 
              kernel = "rectangular")
    cm <- confusion_matrix(fitted(t), iris[folds[[i]], 5])
    a <- accuracy(cm)
    s = s + a
    accs[k, i] = a
  }
  accs[k, nfolds + 1] = s / nfolds
}
rownames(accs) <- paste ("k = ", ks, sep = "")
colnames(accs) <- c(paste("fold ", 1:nfolds, sep = ""), "average")
